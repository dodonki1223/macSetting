# Finderの独自キー設定

Finderに独自のショートカットキーの設定を追加します。

### 設定するキーの一覧

| キー             |  結果                | 
|:-----------------|:--------------------:|
| 新規Folder       |  ctrl + K            |
| ここに項目を移動 |  ctrl + command + V  |
| ゴミ箱に入れる   |  ctrl + d            |
| 情報を見る       |  ctrl + l            |

### 注意事項

画面上は設定されていないように見えますが実際には設定されています。  
設定を反映させるためにはログアウトすること！！
